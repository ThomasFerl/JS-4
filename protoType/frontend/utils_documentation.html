wait(ms)

Wartet eine bestimmte Anzahl an Millisekunden. Gibt ein Promise zurück.

await wait(1000);

buildRandomID(offset)

Erzeugt eine eindeutige ID, basierend auf dem aktuellen Zeitstempel und optional einem Offset.

let id = buildRandomID();

isHTMLElement(element)

Prüft, ob es sich bei element um ein gültiges DOM-Element handelt.

if (isHTMLElement(div)) { ... }

calculate(val1, op, val2)

Führt eine einfache arithmetische Operation aus (+ - * /).

let result = calculate(3, '*', 4); // → 12

formatFloat(value, digits)

Formatiert eine Fließkommazahl mit einer bestimmten Anzahl Nachkommastellen.

let s = formatFloat(3.14159, 2); // → "3,14"
